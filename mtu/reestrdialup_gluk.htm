Глюк опции "Сохранить пароль"

    Известный глюк. Не работает опция "Сохранить пароль"("Save Password") в окне удаленного доступа к сети. Вот что сообщают нам по поводу этого www.winfiles.com:

Опция, доступна но не работает.

    Первое, на что следует обратить внимание, это то, что для активации опции Save Password, Вы должны успешно соединиться с удаленной системой по крайней мере один раз с опцией Save Password в активном состоянии. Это также означает, что, если Вы имеете больше чем один профиль(сеанс) соединения, то Вы должны установить по крайней мере одно успешное соединение с каждым профилем, даже если ваш пароль - одинаков для всех профилей.

Опция не доступна (grayed):

    Это может возникнуть по целому ряду причин, есть много способов которыми Вы можете пробовать и обойти эту проблему. Один из способов состоит в том, чтобы изменить ваше Компьютерное Имя на входе в систему, т.е. ИДЕНТИФИКАТОР, который Вы используете для вашего ISP. Его можно изменить, дважды щелкнув на значке Сеть в Панели Управления, затем измените Ваши данные в разделе Компьютер (вы должны будете перезагрузиться после этого).

    Иногда, переустановка раздела Networking может помоч решить эту проблему. В Панели управления, выберите опцию Programs Add/Remove. Затем идите к разделу Установки Windows, и погасите опцию Networking. Нажмайте OK для закрытия всех диалоговых окон, затем перезагрузитесь. Теперь повторите все шэти шаги с той лишь разницей, что опцию Networking тепрь надо активировать.

    Некоторые пользователи удачно решали эту проблему, устанавливая клиента "Пользователь сетей Microsoft". В Панели управления, откройте раздел Сеть. Если клиента "Пользователь сетей Microsoft" нет в представленном списке, нажмите кнопку добавить, затем выберите Клиент, Microsoft, и затем "Пользователь сетей Microsoft". Нажмайте OK для закрытия всех диалоговых окон, затем перезагрузитесь.

    Последнее, что можно пробовать - удалить Ваш файл пароля. Селайте поиск на файлы * .PWL, затем удалите, или переименуйте их. Перезагрузите вашу систему, и вы обратите внимание, что Windows cпросит о имени пользователя и пароле, чтобы регистрировать Вас опять. Теперь обновите эту информацию о пароле в вашей системе. Вы можете не вводить здесь пароль, если Вы не хотите использовать его, каждый раз при в ходе в систему, но Вы должны ввести имя пользователя. Это восстановит ранее удаленный(переименованный) файл пароля, и может исправить возникшую проблему с опцией Save Password.

От себя добавим, что самым последним способом мы, лично, несколько раз пользовались вполне успешно. Так что стоит начать именно с него :-) Описываемые файлы обычно лежат в основной директории Windows и имеют вид <имя>.pwl где <имя> это то, что вы задавали при регистрации входа в систему. Удачи.
